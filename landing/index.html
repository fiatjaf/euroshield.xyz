Protect your website from GDPR (beta)

Install our JS snippet and block access to your website to any EU-area visitor.

Just place this at the end of your HTML:

<script src="https://euroshield.xyz/shield.js"></script>

Options

  • Append ?ask=y to the script URL to prompt everybody, not only IPs that come from countries on the European Union.
  • Use ?ask=n to block any IP that comes from these countries without asking.

Demo

See how it works by visiting https://alhur.es/ (you should be prompted to answer if you are an EU citizen even if you're not in the EU for demo purposes).

Nothing is stored on the server, localStorage is used to store the answers from the visitors.
